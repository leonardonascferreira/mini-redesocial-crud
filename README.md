# Mini Rede Social - CRUD em JavaScript

Projeto simples de uma **mini rede social** desenvolvido com **JavaScript puro (Vanilla JS)** para demonstrar o funcionamento das operações básicas de um sistema: **CRUD (Create, Read, Update, Delete)**.

A aplicação permite criar posts, visualizar posts, editar conteúdo diretamente na interface e excluir posts.

## 🚀 Funcionalidades

- Criar novos posts
- Listar posts existentes
- Editar conteúdo do post em tempo real
- Deletar posts
- Atualização dinâmica da interface

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Manipulação do DOM
